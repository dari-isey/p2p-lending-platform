\# P2P Lending Platform



Учебный проект по разработке веб-приложения для платформы P2P-кредитования.



\## О проекте



Проект разработан в рамках университетской проектной деятельности.



Пользователь может:



\- зарегистрироваться;

\- авторизоваться;

\- создать заявку на получение займа;

\- инвестировать в заявки других пользователей;

\- просматривать информацию о своих заявках и инвестициях.



\## Используемые технологии



\- Python

\- FastAPI

\- PostgreSQL

\- SQLAlchemy

\- HTML

\- CSS

\- JavaScript
